# UYe-iOS

中腾汇达U业帮，为教育培训机构和学员就业保驾护航。
为培训机构学员提供就业指导服务，监测就业进展，并针对未就业成功学员提供资金帮扶。
促进机构在就业上进行努力，提升整个行业服务水平。
